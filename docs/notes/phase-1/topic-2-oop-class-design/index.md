## :material-target: What You'll Master

- **Classes & Objects** - Instance creation, memory allocation
- **Encapsulation** - Access modifiers, information hiding
- **Inheritance** - Class hierarchy, constructor chaining
- **Polymorphism** - Runtime dispatch, vtable mechanism
- **Abstraction** - Abstract classes vs interfaces
- **Composition** - Favoring composition over inheritance
