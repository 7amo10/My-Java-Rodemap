## Overview

This repository contains a curated collection of Java learning materials, organized as a documentation site built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/). It integrates notes from industry-standard courses, insights from authoritative books like *Effective Java*, and deep dives into Java internals.

### What You Will Find

| Section | Description |
|---------|-------------|
| **Learning Notes** | Structured notes covering Core Java, Advanced Java, and Frameworks |
| **Book Summaries** | Key takeaways from essential Java books (Effective Java, Clean Code, etc.) |
| **Resources** | Curated tutorials, courses, tools, and external references |
| **Projects** | Applied project documentation and case studies |

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/7amo10/My-Java-Rodemap.git
   cd My-Java-Rodemap
   ```

2. **Create a virtual environment** (recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install mkdocs-material
   ```

4. **Start the development server**
   ```bash
   mkdocs serve
   ```

5. **View the site**
   
   Open your browser and navigate to `http://127.0.0.1:8000`

## Deployment

### GitHub Pages

To deploy the site to GitHub Pages:

```bash
mkdocs gh-deploy
```

This command builds the static site and pushes it to the `gh-pages` branch.

## Features

- **Dark/Light Mode** — Toggle between themes for comfortable reading
- **Full-Text Search** — Quickly find content across all pages
- **Responsive Design** — Optimized for desktop, tablet, and mobile
- **Tag-Based Navigation** — Browse content by topic tags
- **Mermaid Diagrams** — Visual representations of concepts and flows
- **Syntax Highlighting** — Code blocks with copy functionality
- **Math Equations** — LaTeX support via MathJax
